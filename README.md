# ScrollingGallery
A little script that creates a simple javascript scrolling gallery that loops indefinitely.

#Usage
Add this snippet somewhere:

```
<script>
new ScrollingGallery({
  'selector': '.selector',
  'speed': 2,
  'width': 150,
  'padding': 20,
  'hoverStop': true,
});
</script>
```

#Requirements:
Requires jQuery to be loaded on the page
